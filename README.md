## Featured Work

### 🧠 AI & Developer Tooling

#### [Vibe-T](https://github.com/sloan-simmons/Vibe-T)
CLI tools for AI agents — designed to execute commands, collaborate with other AI agents, query repositories, use web search, generate plans, automate browsers, and improve terminal-based AI workflows.

**Focus:** TypeScript, AI tooling, CLI systems, automation, developer productivity

#### [Claude-Pad](https://github.com/sloan-simmons/Claude-Pad)
A monorepo containing browser extensions and model-instruction tooling, including Chrome and Firefox extension work for enhancing AI-assisted thinking workflows.

**Focus:** TypeScript, React, browser extensions, Webpack, AI UX

---

### 🔐 Security & Systems

#### [BeaconHunter](https://github.com/sloan-simmons/BeaconHunter)
A Windows security tool that uses Event Tracing for Windows to detect suspicious C2 beacon-like behavior, monitor processes, score network callback patterns, and support thread-level response actions.

**Focus:** C#, Windows internals, ETW, security monitoring, process analysis

---

### 🗂️ Knowledge & Productivity Tools

#### [ObsidianData](https://github.com/sloan-simmons/ObsidianData)
An Obsidian Dataview-style project focused on querying notes with SQL-like syntax, creating data views, working with inline metadata, and extending knowledge-base workflows.

**Focus:** TypeScript, Obsidian plugins, metadata querying, productivity systems

---

### 🏗️ Full-Stack & Backend Architecture

#### [WorkShop](https://github.com/sloan-simmons/WorkShop)
A .NET multi-service architecture with API server, BFF, Identity Server, Device, Scheduler, shared modules, GraphQL, CQRS patterns, Docker support, and test coverage.

**Focus:** .NET, C#, React, GraphQL, CQRS, Docker, microservice-style architecture

---

### 🧩 Desktop & Tooling
